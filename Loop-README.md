<h1>🔰 Loops Overview :</h1><br><br>
<h2>" LEVEL 1 – THE WARM‑UP "</h2>
<i>These are so simple that you should be able to write them blindfolded. If you can’t, you practice until you can. As ans in L1,L2 ..</i>
<hr><br>
<b><h4>1- Print 1 to N
Write a program that reads an integer N and prints all numbers from 1 to N (each on a new line) using a for loop.
Example:</h4> N=5 → 1 2 3 4 5</b>

<b><hr><br>2- Sum of first N natural numbers
Read N. Print the sum 1+2+...+N. Do not use the formula, compute with a loop.
Example: N=5 → 15</b>

<b><hr><br>3- Factorial
Read a non‑negative integer n. Print n! (factorial). Use a loop.
Edge case: 0! = 1.</b>

<b><hr><br>4- Count digits
Read an integer (could be negative). Count and print the number of digits. Use while loop.
Hint: For negative, take absolute value or handle sign. Example: -4502 → 4 digits.</b>

<b><hr><br>5- Table of a number
Input an integer N. Print the multiplication table of N from 1 to 10 in a clean format.
Example: N=3 → 3 x 1 = 3 ... 3 x 10 = 30</b>
